## Installation

<InstallTabs items={['CLI', 'Manual']}>

  <Tab value='CLI'>
  
    ```package-install
    npx @kosori/cli add components input
    ```

  </Tab>

  <Tab value='Manual'>

    <Steps>

      <Step>
        ### Copy-paste the component

        Copy and paste the component code in a `.tsx` file.

        ```json doc-gen:file
        {
          "file": "../../packages/ui/src/input.tsx",
          "codeblock": true
        }
        ```

      </Step>

      <Step>
        ### Update import paths

        Update the `@kosori/ui` import paths to fit your project structure, for example, using `~/components/ui`.

      </Step>

    </Steps>

  </Tab>

</InstallTabs>

## Usage

```ts
import { Input } from '~/components/ui/input';
```

```tsx
<Input />
```
