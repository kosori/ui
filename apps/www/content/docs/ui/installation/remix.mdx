---
title: Remix
description: Installation guide for Remix.
---

<Steps>

  <Step>
    ### Create project

    Run the `create remix` command to create a new React project.

    ```package-install
    npm create remix
    ```

  </Step>

  <Step>
    ### Run the CLI

    Run the `@kosori/cli init` command to configure the project.

    ```package-install
    npx @kosori/cli init
    ```

  </Step>

  <Step>
    ### That's it!

    You can now start adding components or hooks to your project.

    ```package-install
    npx @kosori/cli add components button
    ```

    The command will install the `Button` component to your project. And you can import it in your code.

    ```tsx
    import { Button } from '~/components/ui/button'; // [!code highlight]

    export const Home = () => {
      return (
        <div>
          {/* [!code highlight:1] */}
          <Button>Click me</Button>
        </div>
      )
    }

    export default Home;
    ```

  </Step>
  
</Steps>
