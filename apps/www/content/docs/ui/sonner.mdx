## About

Sonner is built and maintained by [@emilkowalski\_](https://x.com/emilkowalski_).

## Installation

Install the following dependencies and copy-paste the component code in a `.tsx` file.

```package-install
sonner next-themes
```

Add the `<Toaster />` component.

```tsx title='app/layout.tsx
import { Toaster } from '~/components/ui/sonner'; // [!code highlight]

export default const RootLayout = ({ children }) => {
  return (
    <html lang='en'>
      <head />
      <body>
        <main>{children}</main>
        <Toaster /> // [!code highlight]
      </body>
    </html>
  );
}
```

## Usage

```ts
import { toast } from 'sonner';
```

```tsx
toast('Event has been created.');
```
