## Installation

<Steps>

  <Step>
    ### Install the primitive
    
    Install the `@radix-ui/react-dropdown-menu` package.

    ```package-install
    @radix-ui/react-dropdown-menu
    ```

  </Step>
    
  <Step>
    ### Copy-paste the component
    
    Copy and paste the component code in a `.tsx` file.

    ```json doc-gen:file
    {
      "file": "../../packages/ui/src/dropdown-menu.tsx",
      "codeblock": true
    }
    ```

  </Step>

  <Step>
    ### Update import paths

    Update the `@kosori/ui` import paths to fit your project structure, for example, using `~/components/ui`.

  </Step>

</Steps>

## Usage

```ts
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from '~/components/ui/dropdown-menu';
```

```tsx
<DropdownMenu>
  <DropdownMenuTrigger>Open</DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuLabel>My Account</DropdownMenuLabel>
    <DropdownMenuSeparator />
    <DropdownMenuItem>Profile</DropdownMenuItem>
    <DropdownMenuItem>Billing</DropdownMenuItem>
    <DropdownMenuItem>Team</DropdownMenuItem>
    <DropdownMenuItem>Subscription</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```
