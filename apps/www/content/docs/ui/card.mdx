## Installation

<Steps>

  <Step>
    ### Copy-paste the component
    
    Copy and paste the component code in a `.tsx` file.

    ```json doc-gen:file
    {
      "file": "../../packages/ui/src/card.tsx",
      "codeblock": true
    }
    ```

  </Step>

  <Step>
    ### Update import paths

    Update the `@kosori/ui` import paths to fit your project structure, for example, using `~/components/ui`.

  </Step>

</Steps>

## Usage

```ts
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from '~/components/ui/card';
```

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card Content</p>
  </CardContent>
  <CardFooter>
    <p>Card Footer</p>
  </CardFooter>
</Card>
```
