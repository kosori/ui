## Installation

Install the primitive and copy-paste the component code in a `.tsx` file.

```package-install
@radix-ui/react-toggle
```

## Usage

```ts
import { Toggle } from '~/components/ui/toggle';
```

```tsx
<Toggle>Toggle</Toggle>
```
