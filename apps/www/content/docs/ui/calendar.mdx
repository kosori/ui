## About

The `<Calendar />` component is built on top of [React DayPicker](https://daypicker.dev).

## Installation

<InstallTabs items={['CLI', 'Manual']}>

  <Tab value='CLI'>
  
    ```package-install
    npx @kosori/cli add components calendar
    ```

  </Tab>

  <Tab value='Manual'>

    <Steps>

      <Step>
        ### Install the primitive

        Install the `react-day-picker` and `date-fns` packages.

        ```package-install
        react-day-picker date-fns
        ```

      </Step>

      <Step>
        ### Add the calendar component

        The `<Calendar />` component requires the [`<calendar />`](/docs/ui/calendar) component.
        Make sure you have it installed in your project.

      </Step>

      <Step>
        ### Copy-paste the component

        Copy and paste the component code in a `.tsx` file.

        ```json doc-gen:file
        {
          "file": "../../packages/ui/src/calendar.tsx",
          "codeblock": true
        }
        ```

      </Step>

      <Step>
        ### Update import paths

        Update the `@kosori/ui` import paths to fit your project structure, for example, using `~/components/ui`.

      </Step>

    </Steps>

  </Tab>

</InstallTabs>

## Usage

```ts
import { Calendar } from '~/components/ui/calendar';
```

```tsx
const [date, setDate] = useState<Date | undefined>(new Date())

return (
  <Calendar
    className='rounded-xl border shadow'
    mode='single'
    selected={selected}
    onSelect={setSelected}
  />
)
```

See the [React DayPicker](https://daypicker.dev) documentation for more information.
