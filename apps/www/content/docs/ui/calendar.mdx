## About

The `<Calendar />` component is built on top of [React DayPicker](https://daypicker.dev).

## Installation

<Steps>

  <Step>
    ### Install the primitive
    
    Install the `react-day-picker` and `date-fns` packages.

    ```package-install
    react-day-picker date-fns
    ```

  </Step>

  <Step>
    ### Add the button component
    
    The `<Calendar />` component requires the [`<Button />`](/docs/ui/button) component.
    Make sure you have it installed in your project.

  </Step>

  <Step>
    ### Copy-paste the component
    
    Copy and paste the component code in a `.tsx` file.

    ```json doc-gen:file
    {
      "file": "../../packages/ui/src/calendar.tsx",
      "codeblock": true
    }
    ```

  </Step>

  <Step>
    ### Update import paths

    Update the `@kosori/ui` import paths to fit your project structure, for example, using `~/components/ui`.

  </Step>

</Steps>

## Usage

```ts
import { Calendar } from '~/components/ui/calendar';
```

const [date, setDate] = useState<Date | undefined>(new Date())

return (
